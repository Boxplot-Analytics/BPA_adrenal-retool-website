Data Management for Amazon Services

Use this website to interact with the Adrenal database. Navigate to the appropriate page using the sidebar (click the Menu button to open). Here is an overview of what each page does:

  • Viewer - An embedded dashboard used for finding, filtering, & viewing data.
  • Modifier - An embedded application used for finding modifying a single record in the database.
  • Uploader - An embedded application used for uploading a strictly formatted CSV file for bulk insertion to (or bulk modification of) the database.
  • Exporter - A tool for exporting data to Excel, PDF, etc. as needed.
