Data Management for Amazon Services

Adrenal helps companies sell on Amazon. Boxplot understands that Adrenal has large quantities of data relating to these services it provides to its clients. Particularly, many NIS sheets and other datasets primarily in .csv/Excel format. Having the files in this format makes it extremely time consuming and difficult to find particular records or update information. Adrenal is seeking a solution to streamline finding, filtering, and managing the hundreds of thousands of products they oversee.

We are recommending a web-based interface as the final deliverable. This is how Adrenal employees will interact with the data and the database. Web-based interfaces can be used from any browser, on any device, anywhere in the world. Employees will visit a simple website that will contain the following pages:

  • Viewer - This will be an embedded dashboard to be used for finding, filtering, & viewing data. This would look similar to the PDF provided by InsightLeap and could have as many pages as needed for various filtering tasks. Including a page for revenue numbers.
  • Modifier - A page that allows the Adrenal employee to find and modify a single record in the database.
  • Uploader - A form where the Adrenal Employee may upload a completed template .csv file for bulk insertion into the database (or bulk modification of the database).
  • Exporter (optional) - This page would allow Adrenal employees to export data to Excel, PDF, etc. as needed. This is not included in the price breakdown below, but could be added if desired.
